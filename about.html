About Me

Summary

When I was growing up I really enjoyed computers and games, so much so that I ended up making a server for the game Garry's Mod that became very successful holding upto 100 people at a time. Running my server sparked an interest in coding as we needed custom content, so I taught myself some LUA with the help of multiple youtube videos such has <CODE BLUE>'s tutorials on youtube.

While the server did have a good run all things come to an end and after about two years we started losing players and money with this I needed to find a new career so I found the Netmatters SCS Scheme and was luckliy accepted and started working towards a career in Web development.
